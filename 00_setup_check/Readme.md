# Setup Check

## Install Bundler

```bash
~: gem install bundler
~: gem install testgen

```
- Make sure your ruby installation is in your PATH env variable.

## Use bundler

- Generate File Structure

```bash
testgen project PuppyWorkshop
```

- Use Bundler to install the dependencies

```bash
~: cd PuppyWorkshop
~/AdoptAPuppy: bundle install
```
